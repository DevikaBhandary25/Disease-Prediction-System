# Disease Prediction System using Machine Learning

## Project Overview
The Disease Prediction System is a Machine Learning based healthcare analytics project developed using Python.  
The system predicts possible diseases based on symptoms entered by the user.

This project helps in:
- Early disease prediction
- Healthcare data analysis
- Supporting medical decision-making
- Reducing diagnosis time

The system uses multiple Machine Learning algorithms such as:
- Random Forest
- Support Vector Machine (SVM)
- Naive Bayes

The project also includes:
- Exploratory Data Analysis (EDA)
- Symptom correlation analysis
- PCA visualization
- Graph generation
- Disease prediction module

---
